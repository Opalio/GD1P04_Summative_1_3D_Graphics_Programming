Program renders one hexagon in one VAO, EBO, and VBO twice by calculating a new PVM matrix and calling draw elements again
	Hexagons interpolate between two textures
Program renders one quad with a looping animation via sprite sheet

All shapes rendered using GL_TRIANGLES and DrawElements()

Camera object is present

Additional functionality
Able to increase FPS of animation with RMB and decrease with LMB
Able to move camera around (translating objects) with WASD
